Get JPSreport

  • Download the code from the Gitlab repository
  • or get the executables from the latest release

Requirements

The following software is necessary to run JPSreport:

  • CMake
  • A compiler with support for C++11 e.g. g++, clang or Visual Studio 2013
  • Boost-libraries. At the moment boost is hard-linked in JPSreport.
  • Python. At least modules Numpy, Matplotlib, Polygon should be installed, since some python scripts are embedded for plotting diagrams.
