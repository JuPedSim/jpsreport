Step 1:

  • Download and install the [Eclipse] (http://www.eclipse.org/downloads/?) platform (here we take the version Juno Service Release 2 as an example).
  • Download [JPSreport source files] (https://cst.version.fz-juelich.de/jupedsim/jpsreport)
  • Install latest version (g++ version should not be older than g++ 4.8.2) of MinGW

Step 2:

Open Eclipse –> File –> New –> C++ Project, give Project name and select Project type and Toolchains as shown in the following figure (MinGW GCC or Cross GCC are all ok). For Location set as the directory where the source files are.

Step3:

  • Right-click on the project and click Properties in the pop-up menu, you will get a window as the figure below.
  • Click C/C++ Build –> Enviorment, please to be sure that the path of GCC and make.exe (for example “./MinGW/bin” if you installed MinGW) is included in “PATH”.
  • Click C/C++ Build –> Settings, Click Tool Settings tab, GCC C++ Compiler –> Dialect, for the “Language standard” please select “ISO C++11”. GCC C++ Compiler –> Includes, add “include paths (-I)” by clicking the button and in the coming dialog box click the button “Workspace” and select the location of the project. GCC C++ Compiler –> Miscellaneous, add the text “-std=c++11 -fopenmp -Wno-unused-local-typedefs” in the textbox of Other flags. Click MinGW C++ Linker –> Libraries, add the text “wsock32” to Libraries (-l) in the right box. Click MinGW C++ Linker –> Miscellaneous, add the text “-fopenmp” in the textbox of Linker flags.
  • Select [All configurations] from the drop-down menu of Configuration and then push the button Apply and OK

Step 4:

  • Exclude the folder Utest from Build by right click on it as shown in the following figure
  • Build and Run JPSreport
