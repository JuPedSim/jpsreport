JPSreport is a computer program that can be used to analyze pedestrian trajectories from experiment. or simulations. It is a C++ program that reads input parameters from a text file (.XML format), calculates basic pedestrian movement parameters like density, velocity and flow, and writes user-specified output data to files. This section describes how to obtain JPSreport and how to use it.

How to Get JPSreport

Detailed instructions on how to download executables, manuals, source-code and related utilities, can be found on the JuPedSim Website. The typical JPSreport distribution consists of an compressed archive, which is available for MS Windows, Mac OS X, and Linux.

Computer Operating System and Software Requirements

The goal of making JPSreport publicly available is to provide researchers and students a tool for analyzing their trajectories obtained from model simulations as well as experiments easily. Thus, JPSreport has been designed for computers running Microsoft Windows, Mac OS X, and various implementations of Unix/Linux.

The following software is necessary to run JPSreport:

  • CMake
  • A compiler with support for C++11 e.g. g++ or clang or Visual Studio 2013
  • Boost-libraries. At the moment boost is hard-linked in JPSreport.
  • Python. At least Modules Numpy, Matplotlib, Polygon should be installed, since some python scripts are embeded for plotting diagrams.
